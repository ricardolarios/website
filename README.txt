Figure out how to convert from SCSS to CSS:
	While making css changes, you can use: 
\f1\fs22 \cf2 \cb3 \CocoaLigature0 sass --watch [scss file path]:[css destination path]
	which will continue to generate css files from the scss file you edit.
	For example: sass --watch styles/scss:styles/css
